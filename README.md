# Smart Campus Management System

A Java console application that simulates how a 
university manages its daily campus operations.

## Modules
- 🏠 Hostel Allocation — Queue + Array
- 📅 Event Booking — ArrayList

- Array — Hostel room management
- ArrayList — Event bookings

2. Open in IntelliJ IDEA
3. Run Main.java
